# 自定义功能使用指南

## 新增功能 (v1.1.0)

### 🔍 自定义搜索引擎

现在您可以选择自己喜欢的搜索引擎：

1. **打开设置**
   - 点击右上角的设置按钮（齿轮图标）
   - 或点击"Quick Links"旁边的编辑图标

2. **选择搜索引擎**
   - Google（默认）
   - Bing
   - DuckDuckGo
   - Baidu（百度）
   - Custom（自定义）

3. **自定义搜索引擎**
   - 选择"Custom"选项
   - 输入搜索URL，使用 `%s` 作为查询占位符
   - 例如：`https://www.ecosia.org/search?q=%s`

### 🔗 自定义快捷链接

完全掌控您的快捷链接：

1. **编辑链接**
   - 点击设置按钮或"Quick Links"旁的编辑按钮
   - 修改图标、名称或URL
   - 图标可以使用任何emoji（最多2个字符）

2. **添加新链接**
   - 在链接编辑器中点击"+ Add Link"按钮
   - 填写图标、名称和URL
   - 支持添加无限数量的链接

3. **删除链接**
   - 点击每个链接右侧的"×"按钮
   - 立即删除不需要的链接

4. **保存更改**
   - 编辑完成后点击"Save Settings"
   - 所有设置自动同步到您的Google账号

5. **重置为默认**
   - 点击"Reset to Default"
   - 恢复初始的6个默认链接和Google搜索

## 功能特点

✅ **云同步** - 设置自动同步到所有登录了相同Google账号的Chrome浏览器
✅ **实时预览** - 保存后立即查看更改效果
✅ **直观编辑** - 简单易用的图形界面
✅ **灵活自定义** - 支持任意数量的快捷链接
✅ **多语言支持** - 支持中英文界面

## 故障排除

**Q: 设置没有保存？**
A: 确保您已登录Chrome并启用了同步功能。

**Q: 快捷链接不显示？**
A: 检查URL是否正确，必须以 `http://` 或 `https://` 开头。

**Q: Emoji不显示？**
A: 确保使用的是标准emoji字符，某些特殊符号可能不兼容。
